## 快速开始

### 方法一：直接打开
1. 克隆或下载项目文件
2. 在浏览器中直接打开 `index.html` 文件

### 方法二：本地服务器
```bash
# 使用Python启动本地服务器
cd fengkeyun-website
python3 -m http.server 8080

# 或使用Node.js
npx http-server -p 8080

# 然后在浏览器访问
# http://localhost:8080
```
